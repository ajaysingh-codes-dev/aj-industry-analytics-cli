# AJ Industry Analytics CLI

## 📌 Project Overview
This is a data cleaning and analytics pipeline built using Python, Pandas, and MySQL.

## ⚙️ Features
- Data cleaning pipeline
- Missing value handling (median, mode, forward fill)
- ID validation (separates incomplete records)
- Logging system for filled values
- Summary and missing report
- Excel report generation

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- MySQL
